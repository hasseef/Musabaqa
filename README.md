# منصة مسابقة — نسخة احترافية كاملة
## الدخول التجريبي
- اسم المستخدم: **admin** — كلمة المرور: **M1234**
- اختر **نوع الحساب** و**الدور** من نافذة الدخول.

## أهم الميزات
- **الشعارات**: شعار مسابقة + تلبية + رؤية 2030 مضمّنة في الهيدر والفوتر.
- **استجابة عالية** للجوال والكمبيوتر والأجهزة الذكية (PWA + Bottom Nav + Dark Mode).
- **رعايات هجينة**: سوق رعايات موجهة لمسابقات تحتاج تمويل + باقات عامة في مركز الرعاة.
- **تمويل ذاتي**: تمويل الجوائز مباشرة من محفظة المنشئ بدون طرح للرعاية.
- **نسبة تشغيل ≤5%** محسوبة تلقائيًا في إنشاء المسابقات.
- **تحكيم موزون**، **استلام جوائز** كامل، و**محفظة** لإدارة الفواتير.
