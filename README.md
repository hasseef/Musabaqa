# منصة مسابقة — إصدار الجوال الاحترافي (SPA + PWA + Roles + Media + Gated MCQ)

## دخول تجريبي سريع
- اسم المستخدم: **admin**
- كلمة المرور: **M1234**
- اختر الدور (مشارك/محكّم/مدير) من القائمة قبل الضغط على "متابعة" للتوجّه لمنطقة الدور مباشرة.

## الميزات
- RTL عربي، Bottom Nav للجوال، Dark Mode
- تسجيل دخول تجريبي حسب الدور مع إعادة توجيه ذكي
- مسابقات تدعم **فيديو** و**ملصقات**
- نموذج تقديم مع **أسئلة اختيار من متعدد** + **تصحيح الإجابات**
- خيار **رمز من مرفق المنظّم** لفتح الأسئلة
- منطقة **التحكيم** بمعايير وأوزان
- منطقة **الإدارة** لإنشاء مسابقات وأقسام وأسئلة بسهولة + تصدير CSV
- **PWA**: تثبيت كتطبيق + Offline Cache
